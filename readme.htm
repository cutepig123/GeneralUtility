2.3      程序执行流程之WinMain

WinMain

         new了一个CFrameWindowWnd，

         调用Create方法创建窗口

                   RegisterSuperclass

                            RegisterClassEx，回调函数为CWindowWnd::__ControlProc

                   RegisterWindowClass

                            RegisterClass，回调函数为 CWindowWnd::__WndProc

                   CreateWindowEx

         然后CPaintManagerUI::MessageLoop进入主循环

                   在CPaintManagerUI::MessageLoop里面调用CPaintManagerUI::TranslateMessage处理消息。

                            TranslateMessage在收到消息LPMSG后,看看该消息句柄是否在注册的窗口里面，如果是就调用该窗口的PreMessageHandler

 

 

关于CWindowWnd::__WndProc

         消息处理总函数

         第一个消息是WM_NCCREATE，在这里他调用SetWindowLongPtr设置userdata为thsi

         对其他所有消息，调用GetWindowLongPtr获取this，

                   如果消息为WM_NCDESTROY

                            调用CallWindowProc（pThis->m_OldWndProc）

                   否则调用pThis->HandleMessage

 

 

2.4      CPaintManagerUI

 

全局一共几个CPaintManagerUI对象呢？

         一部分是全局的，全局的主要是几个static函数，如SetResourceInstance

         一部分是跟着窗口的

 

 

CPaintManagerUI::MessageHandler实现了一个默认的消息处理函数

         处理m_aMessageFilters

         Custom handling of events，包括WM_CLOSE，WM_ERASEBKGND，WM_PAINT，WM_PRINTCLIENT，WM_SIZE,WM_TIMER,WM_MOUSELEAVE，WM_MOUSEMOVE，WM_LBUTTONDOWN，

         处理WM_CLOSE

                   Make sure all matching "closing" events are sent，包括UIEVENT_MOUSELEAVE，UIEVENT_BUTTONUP，focus处理

         处理WM_PAINT

                   调用GetUpdateRect获取更新区域

                   处理WM_SIZE

                            创建m_hDcPaint

                            移动m_pRoot位置

                            发送windowinit

                   Set focus to first control?

                   Render screen

                            处理m_anim（for 3d animation）

                            处理Standard painting of control-tree - no 3D animation now

                                     处理 offscreen bitmap

                                     调用 m_pRoot->DoPaint

         处理WM_TIMER

                   对m_aTimers中句柄为m_hWndPaint的发送UIEVENT_TIMER消息

         处理WM_MOUSEHOVER

                   发送UIEVENT_MOUSEHOVER事件

                   Create tooltip information，包括创建tooltip窗口，发送TTM_SETTOOLINFO，TTM_TRACKACTIVATE消息

         处理WM_MOUSEMOVE

                   Start tracking this entire window again by _TrackMouseEvent

                   Generate the appropriate mouse messages

                            找到当前control by FindControl

                            如果control改变，发送UIEVENT_MOUSELEAVE，UIEVENT_MOUSEENTER消息

                            如果当前已经按下鼠标，发送UIEVENT_MOUSEMOVE
