Thanks God，It’s Friday

分类：闲心碎语 ｜ 作者：酷~行天下 ｜ 发表于2011/12/02 4条评论 1,236 views

     12月，终于没蚊子了， 不过，我也快被冻死了。

     11月终于把《算法导论》（CLRS）看完了，从7月27号开始发 CLRS 第一篇笔记，到11月27号最后一篇，整整4个月时间，比预期的时间要短，可能是后边算法基本都学过的缘故。现在想想，都觉得诧异，竟然把CLRS看完了，而且看完的是“我”，这要在半年前，打死我都不会相信，因为无法想象。普通人可能没什么感觉，如果是计算机行业 + 接触过算法，十有八九会说“卧槽，这货竟然把算法导论看完了！”。不过现在回想，4个月，怎么会这么长，看书倒是不费时间，更多的是花在代码实现上。并不是全天候学一本书，而且有两次，连续十几天都没看，所以，此书其实没传说中的那么难。

     CLRS并不完美，不少算法可能是迫于篇幅，我是没看明白，所以学习一个算法时，通常是《算法导论》+《数据结构与算法分析》 + 《数据结构》（此书不推荐，代码风格很烂） + 《数据结构》（更不推荐，必然让你走弯路的一本书） + Google 才搞懂。

     当初看斐波那契堆时，参考CSDN上某文，那位仁兄原话“连纸上研究，再代码实现……加起来大概用了20小时。只比这个时间多，不比这个时间少了。”20个小时，大概是正常时间，悲催，我当时死磕了整整三天才搞明白。不过好的结果是，现在的那篇《斐波那契堆》笔记，目前为止，一直是点击率最高的一篇。

                                            

     未鹏大牛认为，学习算法，最重要的是去研究这个算法从无到有的过程，而不是现成摆在那的结果，因为前者是学习能力，思考能力的进步，而后者仅仅是知识的扩充。这个道理浅显易懂，不过，我依然是按照后者的标准来学习的，学习算法本身而不是去研究它的来由。虽然前者很可贵，但不是我看此书的初衷，当初决定看CLRS，原因很简单，算法零零总总学了一年，应该系统的整理一下，于是现在目的达到了，压根没想过再去研究本《TAOCP》，目的不同，无所谓褒贬。

     未鹏大牛：“我有一个习惯，看定理必看证明。”可惜，看CLRS过程中，我看定理很少看证明（但并不是不看），因为算法很多定理 ≈ 数学定理，而看数学定理，在我眼里，说白了，就是死磕。可能有人要说了，数学思维，数学××是多么多么重要，你试过看数学定理吗，试过研究一个数学定理的证明，花费两三个小时吗，我试过，唯一的感觉：得不偿失。后来就很少看了，我是来学算法的，不是来搞数学证明的，而且经常，某个定理对整个算法来讲只是一个微小辅助，因为一个完整的算法，最核心的思想，通常和数学扯不上什么关系。

     算法扯的有点多了，虽然学的还不够深入，但是很满足了。12月开始看《深入理解计算机系统》（CSAPP）和《操作系统设计与实现》，前者697页，后者428页。目前的进度是，每天35页 + 每天20页，除去杂事干扰，譬如期末考试之类，大概12月结束时可以看完。CSAPP是个惊喜，好比冬日里的阳光，此书比预想的要有趣很多，甚至让我改变了对“课后习题”这个万恶之首的看法，很多习题极具诱惑力，让我忍不住去看，而且解答通常让人拍案叫绝。而写出MINIX操作系统的老教授， Tanenbaum 大叔的 OS 一书比 CLRS 还苦逼，看不懂 + 看不懂，即便去年已经修过一门叫做《操作系统》的专业核心课，好在昨天在此书95页看到两句话：

 

     “ 它确实使人们想起 Ken Thompson 在早期 UNIX 的源代码中所写的一句著名的注释：

     /*You are not expected to understand this */                       ”

     好吧，Tanenbaum 大叔没指望我看懂，"You are not expected to understand this "，那我就先溜一遍再说，看不懂吗？ 保不准到后边就会理解前边的，或者，某天突然就明白了，N天之前看过的某段话是什么意思。通常都是这样干的。

                                                 

__________________________________________________________________________________________________________

12月3号横插一腿：

     昨天下午看Tanenbaum大叔书中的 MINIX的中断处理，MINIX3的进程通信，MINIX的进程调度，扛不住了，理论讲的太少，对我来说，过于偏实践了，晚上亚马逊订了两本书《现代操作系统》和《Orange’s 一个操作系统的实现》，一本纯理论，一本纯实践， 三本正好互补。

                                                 

     早上9点，书到手（亚马逊和京东的效率真TM高），《现代操作系统》翻了三四十页，貌似正是需要的书，通俗 + 严密。至于另一本，以前在图书馆翻过几章，实践操作的好书。看来操作系统和算法一样，不是一本书能打发掉的，那就一起来吧。_________________________________________________________________________________________________________

     前两天看到一句话“人的一生只有 900 个月”。算了一下，保不准连 900 个月都不到， 而我现在最值钱的就“时间”二字，这学期干过的也就一件事，投资时间，过去几个月，投资的还不错，除了小方向的书有些偏差。12月准备重点投资给上边说过的两本书，而这种情况通常会引发一个问题，“内境逾宽、外延逾窄”，大二确实是这么过来的，几乎没有外延，不过身不由己。专业专到只知道专业，那就杯具一出了，成了井底之蛙都不自知。现在愤青特别多，只有一个原因：知识过于局限，不能理解为什么这个世界是现在这个样子，就开始胡乱抨击了。

     这学期一开始就试图保持平衡，自然科学 + 人文社科 ≈ 1 + 1，不知道这样理解 拉里•佩奇 说的“Be an expert in all aspects.”是否正确，但最起码，多学科学习真的受益匪浅。不过过去一两个月专业学习进度有点慢了，12月份打最后一仗，如果能结束 CSAPP 和 OS 两本书，这学期就算完美了。C，算法和数据结构，操作系统，系统结构。最基础的几样先搞定，下学期就可以放开来学更多扩展技术知识了，譬如 Linux + Unix + 附带的各种知识（Vim之类），甚至，Java代码一直敲，却发现越敲越不会，越敲越水，《Java编程思想》在向我招手，不过，贪多嚼不烂，下学期学。

     今天星期五，不用逃课，因为没有课。Thanks God，It’s Friday。这学期要结束了，专业课依然保持惯例：“一节不听，能翘则翘”。大一时，心理学老师 杨甦红（一个哲学经济学博士） 说：“教育是一种启迪，是教会人举一反三”，翘课才可以有更多的时间去举一反三，所以，翘课，多么有意义啊！

     11月15号，去上两三周没去过的数据库课。

                 职业：数据库课大BOSS，花衣魔笛手

                 装备：WinTel微机，黑板，点名册，超声波

                 攻击：音波和本人共振，堪称魔音，攻击力 INT_MAX（0x7fffffff）

                 技能：SQL，Angry Birds，×××你到黑板上解这个题：

                 大招：咦，你怎么还带耳机啊，到第二排来坐。

    第一节课下课时，最后一排，我在看书，突然感觉身旁有个人，一抬头，擦，大BOSS巡视，后续情节却出乎意料，“哇塞，像个世外桃源，这么悠然自在啊。”我说是啊。诡异，神奇，从那之后，和大BOSS就这样和解了，再也不管我是不是带耳机，是不是坐最后一排，是不是不听课，甚至上课时只管别人，不管我了，瞟一眼然后往回走，还有什么比这更好的事。下课下来问我看什么书，从侧面看到了，然后说“《黑客》，只听过，看看就可以了”。貌似这学期最难熬的一个老师，终于不用熬了。

     前几天东东给我一个链接，一个叫 海洋 的家伙，大四 + 腾讯，21周岁，相比之下，我弱爆了，不接触各种各样的人，无法想象世界之大。方建中上课时，说过的一段话，大致意思是“世界上只有两种人，一种是老鹰，一种是蜗牛，这两种从出生就注定一个在天空，一个在地上，是完全不同的动物，它们惟一相同的是都能到达金字塔顶。”老鹰在哪里，在斯坦福，在CMU，在MIT，在清华，在浙大，集天下英才而教之。蜗牛在哪里，很明显，在这里，我就是蜗牛，而且是觉醒很晚的蜗牛，想要到达金字塔顶，想要达到和老鹰类似的高度，唯有永不停息的执着了。

老鹰Sample：

1）克雷格•西尔弗斯坦（Graig Silverstein）几乎一个人写出了Google搜索引擎的第一个商业版本。（当时佩

奇和布林忙于融资，西尔弗斯坦是Google早期几乎唯一工程师）

2）朱光灿，中国工程师，美国拿计算机博士，一个人推出了Google的图片搜索，这不光包括研究算法写程序，

而且包括架设和管理服务器等杂活。

     一个一流的工程师，胜过几百个二流工程师，差距就是这么明显，而我，还在金字塔底，努力 + 慢慢爬吧。

                     

    PS：此文有点碎碎念，不管了，自习去了。上边的蜗牛吃点儿包心菜，爬起来可能快点。
